FMDomMarker

FMDomMarker 自定义dom元素标注对象，为自定义图层。

new fengmap.FMDomMarker(options)

Name Type Description
options json
Name Type Description
height number

标注高度, 默认 1
position json

位置

Name Type Description
x number

图片地图位置x值，默认对应楼层中心点坐标
y number

图片地图位置y值，默认对应楼层中心点坐标
fontsize number

文本标注字号
domContent string

标注dom元素，也可直接填写文本。
domWidth number

标注dom元素的宽度。
domHeight number

标注dom元素的高度度。
anchor fengmap.FMMarkerAnchor

标注锚点，默认为 fengmap.FMMarkerAnchor.CENTER
