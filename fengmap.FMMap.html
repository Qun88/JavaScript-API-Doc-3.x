Class: FMMap

FMMap

FMMap 是加载、显示、操作地图的主类，一个页面中至少需要包含一个FMMap对象

new fengmap.FMMap(appName, key, id, container, options)

构造FMMap类

Name Type Description
appName string

地图应用名称，通过蜂鸟云后台创建
key string

地图应用密钥，通过蜂鸟云后台获取
id string

地图id，通过蜂鸟云后台获取
container dom

FMMap 地图容器
options fengmap.FMMapOptions
