Class: FMGradientPolygonMarker

FMGradientPolygonMarker

FMGradientPolygonMarker 自定义渐变多边形标注对象，为自定义图层。

new fengmap.FMGradientPolygonMarker(options)

Name Type Description
options json

参数

Name Type Description
alpha number

多边形的透明度。0.0 - 1.0 之间取值, 默认为 1.0
color number

多边形的颜色,默认 #FF0000
height number

离地高度, 默认为 1
points array

多边形顶点
points[].x number

x坐标
points[].y number

y坐标
scale number

缩放，默认为 1.0
scaleValue number

缩放值
